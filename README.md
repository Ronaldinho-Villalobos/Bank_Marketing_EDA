# 🏦 advanced-bank-marketing-analytics-dashboard
## 📊 Advanced Exploratory Data Analysis with Streamlit

---

## 📝 Descripción

Aplicación analítica desarrollada con **Streamlit** que implementa un Análisis Exploratorio de Datos (EDA) avanzado sobre el dataset de campañas de marketing bancario.

Este proyecto demuestra la aplicación práctica de:

- ✅ Python Fundamentals  
- ✅ Análisis Exploratorio de Datos (EDA)  
- ✅ Programación Orientada a Objetos (POO)  
- ✅ Visualización interactiva  
- ✅ Dashboards dinámicos  

El sistema permite analizar qué factores influyen en la aceptación de campañas bancarias mediante métricas estadísticas, visualizaciones dinámicas y segmentación interactiva.

---

## 🎯 Objetivo del Proyecto

Identificar patrones y variables clave que impactan la probabilidad de que un cliente acepte una campaña de marketing bancario.

Se busca:

- Comprender el comportamiento del cliente  
- Detectar variables influyentes  
- Analizar patrones demográficos y económicos  
- Generar conclusiones estratégicas

## 📊 Funcionalidades del Dashboard

### 🏠 1. Home
- Presentación del proyecto  
- Objetivo del análisis  
- Información del autor  
- Tecnologías utilizadas  

### 📂 2. Carga de Datos
- Subida dinámica de archivo CSV  
- Vista previa del dataset  
- Métricas generales (filas y columnas)  
- Renombrado de variables con alias entendibles  

### 📊 3. Análisis Exploratorio (EDA)

#### 1️⃣ Información General
- Tipos de datos  
- Valores faltantes  
- Estadísticas descriptivas  
- Tasa de aceptación (%)  

#### 2️⃣ Distribuciones Numéricas
- Histogramas interactivos  
- Boxplots  
- Comparación por variable objetivo  

#### 3️⃣ Variables Categóricas
- Frecuencias  
- Comparación por aceptación  
- Visualización agrupada  

#### 4️⃣ Análisis Bivariado
- Numérica vs Variable Objetivo  
- Boxplots comparativos  
- Identificación de patrones  

#### 5️⃣ Análisis Dinámico
- Filtros por rango de edad  
- Segmentación interactiva  
- Actualización automática de gráficos
